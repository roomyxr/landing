# Roomy | The Future of Spatial Productivity

Roomy is a next-generation spatial workspace designed for Meta Quest, Steam VR, and mobile AR. We are building the digital sanctuary for focus, immersive learning, and mental balance.

## 🚀 Built With
- **Next.js 15** (App Router)
- **Tailwind CSS** (for that Lo-Fi aesthetic)
- **Framer Motion** (for buttery smooth spatial animations)
- **TypeScript** (type safety for scalability)

## 🌐 Quick Links
- **Website:** [Roomy-Yhe Learning Room](https://roomyxr.com)
- **Community:** [Join our Discord](https://discord.gg/uSvJVJ3m6))

## 🛠 Tech Stack
- **Frontend:** React + Next.js
- **Styling:** TailwindCSS
- **Deployment:** Vercel (CI/CD Pipeline)
- **Analytics/CRM:** Discord Webhook Integration
